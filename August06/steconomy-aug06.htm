Economic Impact

The Economic Impact provides a quantifiable and economically sound method for determining the relative impact of proposed or impending changes to a regional economy. It allows users to discover which sectors of the local region actually drive the economy.

State of Economy August 2006

The State of Economy allows users to view an in-depth report on the state of the economy as of 2005. 

Economic Base

Economic Base displays which NAICS super-sectors drive the economy.  The data columns show the number of jobs in each sector, the total earnings of each sector, and the percentage of jobs that each sector is responsible for, as well as average annual earnings per worker.

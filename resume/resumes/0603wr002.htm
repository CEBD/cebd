I have designed all of Cyan’s products.    In reverse chronological order, these products include:

 

'PRP'  Propeller Pitch Controller (Aircraft industry):   A microcomputer‑based controller which measures and regulates the pitch (and therefore the RPM) of propellers on small aircraft.

 

'SPC'   Solar Pump Controller (Solar Power industry):   A microcomputer‑based controller which connects between an array of solar panels and a water pump.     It monitors the solar power, computes the correct power‑impedance match between the panels and the pump, and delivers the conditioned power to the pump.    The SPC exceeds the capability of competitors' products because it requires no adjustments and is very efficient.   (Customers often remark about its ability to track and maintain control in reaction to clouds passing overhead.)

'Portfinder' (Computer industry):  The Portfinder allows its user to quickly and easily identify the port numbers of a computer's RS232 serial and parallel printer ports.

 

'RCAT'  Ribbon Cable Assembly Test Unit (Computer industry):   The RCAT provides quick and easy tests of IDC (Insulation Displacement Connector) cable assemblies.  

 

External Floppy Disk Drives (Computer industry):   Between 1990 and 1995, we produced and marketed external floppy disk drives to individuals, companies, and distributors.   They were standard floppy disk drives, coupled with our case, power supply, cables, and installation software.  We produced over 60 different models.   Because of changes within the industry, these products have been discontinued.

 

'CRAFT'  Craft Store Software Program:    This is a complete database  of Accounting, Inventory, and Point of Sales software designed specifically for Craft stores;    it meets the unique requirements of Craft stores, whose ownership rents space to (and also manages and tracks all the retail sales of) multiple vendors.

 

'RECORDER' Software Program (Manufacturing):   The 'Recorder' software program allows a low‑cost computer to perform the function of an industrial data recorder.    Real-time count or time data is taken from up to 256 channels of the assembly line,  with 1 millisecond resolution, and logged for periods as long as  6 months.  (Data is normally downloaded daily.)    Each I/O point of the assembly line can be either 220VAC, 110VAC, 24VAC, 24VDC, 12VDC, or 5VDC.    This program is a full-featured production logging system that allows production, maintenance, and quality control  personnel to acquire and graph machine performance data, including count totals, run times, etc., from any assembly line.

 

'PM' Preventative Maintenance Software Program (Manufacturing):   The 'PM' software program is similar to 'Recorder' except that its features are designed to inform maintenance personnel of the maintenance status of the various subassemblies of the automated assembly line.     It  includes operational set points so that maintenance alarms are triggered when user-specified run times or production totals are reached.  Using this system, maintenance personnel can develop a custom designed, fully integrated, color coded, password protected,  preventative maintenance schedule for each subassembly of their assembly line.

 

'RS232'  Serial Communications Software Program (Machine‑Tool industry): The RS232 software program is a generic tool for serial communications between the computer and any other electronic machine.  We market several specialized versions of it to the Numerical Control (NC) Machine Tool industry ‑ e.g., automatic mills, lathes, etc.    Its intuitive format allows quick and easy configuration of high speed, error‑free communications between machines.  Its features provide machine tool programmers and production personnel (who are not always experienced with downloading NC programs from the computer into their machines) a means of straightforward, easy‑to‑use, production downloading.

File Conversion Software Programs (Manufacturing):    In conjunction with the use of the RS232 software, several custom file conversion programs were developed at the request of manufacturers for use on their specific specialized manufacturing equipment.

 

 

Random Sampler (Manufacturing):   The Random Sampler is a unique industrial instrument which automates and completely randomizes the quality assurance sampling of items from a high‑speed production line.   Its basic operational algorithm is based upon a concept originated by SWOSU physicist Dr. Ray Jones.

 

 

Power Monitor (Manufacturing):   The 'Power Monitor' is a digital, microcomputer-based test instrument which displays the power (watts) and energy output (joules) of industrial Ultrasonic Welders.

 

 

'INTRPT'  Computer Interrupt Testing Software Program (Software development industry):   This is a utility software program which is used by computer programmers to test the results of software interrupts.

 

 

'BASLIN'  Basic Language Line Numbering Software Program (Software development industry):    This is a utility software program which allows computer programmers who write programs in the Basic programming language to write longer, more elaborate programs.
